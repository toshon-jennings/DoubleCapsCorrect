To confirm the authenticity of CAps™ v2.1.1, I provide:

  • 🗝️ A public key: CAps_public.asc
  • 🔏 A signature file: CAps_v2.1.1.exe.sig
  • 📄 A hash list: Verify_Hashes_2.1.1.txt

Step 1: Import the Public Key

Use GPG or Kleopatra to import CAps_public.asc.

Step 2: Verify the Signature

Run this command with GPG:

gpg --verify CAps_v2.1.1.exe.sig CAps_v2.1.1.exe

Step 3: Confirm the Hash

Run one of the following to check the file integrity:

certutil -hashfile CAps_v2.1.1.exe SHA256
shasum -a 256 CAps_v2.1.1.exe
